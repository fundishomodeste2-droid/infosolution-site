# Infosolution Site

Projet React + Vite + Tailwind.

## 🚀 Développement

```bash
npm install
npm run dev
```

## 📦 Build

```bash
npm run build
npm run preview
```

## 🌍 Déploiement sur Vercel

1. Poussez ce projet sur GitHub.
2. Connectez le repo à Vercel.
3. Vercel détectera Vite automatiquement.
